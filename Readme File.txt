# Global Superstore Sales Dashboard (Power BI)
==========================================================================================================================================================================================
## Project Overview

This project presents an interactive Power BI dashboard built using the Global Superstore dataset. The dashboard provides insights into sales performance, profit trends, customer activity, regional performance, and market analysis.
==========================================================================================================================================================================================
## Objectives

* Analyze overall business performance.
* Track sales and profit trends over time.
* Identify top-performing categories and markets.
* Monitor key business KPIs.
* Support data-driven decision making.

## Key Insights

1. Technology category generated the highest sales ($4.7M).
2. APAC is the top-performing market.
3. Sales increased significantly between 2016 and 2019.
4. Profit reached approximately $0.5M in 2019.
5. The dashboard enables quick identification of high-performing regions and categories.
